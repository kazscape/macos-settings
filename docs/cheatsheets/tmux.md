# tmux チートシート

## プレフィックスキー

- **Prefix**: `Ctrl-a` (デフォルトの `Ctrl-b` から変更済み)

## セッション管理

| キーバインド | 説明 |
|------------|------|
| `tmux` | 新しいセッションを開始 |
| `tmux new -s <name>` | 名前付きセッションを作成 |
| `tmux ls` | セッション一覧を表示 |
| `tmux attach -t <name>` | セッションにアタッチ |
| `Prefix d` | セッションからデタッチ |
| `Prefix $` | セッション名を変更 |

## ウィンドウ管理

| キーバインド | 説明 |
|------------|------|
| `Prefix c` | 新しいウィンドウを作成 |
| `Prefix ,` | ウィンドウ名を変更 |
| `Prefix &` | ウィンドウを閉じる |
| `Prefix n` | 次のウィンドウに移動 |
| `Prefix p` | 前のウィンドウに移動 |
| `Prefix 0-9` | 指定番号のウィンドウに移動 |
| `Prefix w` | ウィンドウ一覧を表示 |

## ペイン管理

### 分割

| キーバインド | 説明 |
|------------|------|
| `Prefix \|` | 垂直分割 (カスタム) |
| `Prefix -` | 水平分割 (カスタム) |

### ナビゲーション

| キーバインド | 説明 |
|------------|------|
| `Prefix h/j/k/l` | vim風にペイン間を移動 (vim-tmux-navigator) |
| `Ctrl-h/j/k/l` | vim/tmux間をシームレスに移動 (vim-tmux-navigator) |
| `Prefix o` | 次のペインに移動 |
| `Prefix ;` | 前のペインに移動 |
| `Prefix q` | ペイン番号を表示 (番号入力で移動) |

### リサイズ

| キーバインド | 説明 |
|------------|------|
| `Prefix h` | 左にリサイズ (5行単位) |
| `Prefix j` | 下にリサイズ (5行単位) |
| `Prefix k` | 上にリサイズ (5行単位) |
| `Prefix l` | 右にリサイズ (5行単位) |
| `Prefix m` | ペインを最大化/元に戻す (カスタム) |
| `Prefix z` | ペインをズーム (最大化/元に戻す) |

### その他

| キーバインド | 説明 |
|------------|------|
| `Prefix x` | ペインを閉じる |
| `Prefix !` | ペインを新しいウィンドウに移動 |
| `Prefix {` | ペインを左に入れ替え |
| `Prefix }` | ペインを右に入れ替え |
| `Prefix Space` | レイアウトを切り替え |
| `Prefix Alt-c` | 現在のディレクトリでセッションにアタッチ (カスタム) |

## コピーモード

| キーバインド | 説明 |
|------------|------|
| `Prefix [` | コピーモードに入る |
| `v` | 選択開始 (vi mode) |
| `y` | 選択範囲をコピーしてクリップボードに保存 |
| `Enter` | コピーモードを終了 |
| `q` | コピーモードを終了 |
| `Space` | 選択開始 (デフォルト) |
| `/` | 下方向に検索 |
| `?` | 上方向に検索 |
| `n` | 次の検索結果 |
| `N` | 前の検索結果 |

### コピーモードでの移動

| キーバインド | 説明 |
|------------|------|
| `h/j/k/l` | カーソル移動 (vi mode) |
| `Ctrl-u` | 半ページ上にスクロール |
| `Ctrl-d` | 半ページ下にスクロール |
| `Ctrl-b` | 1ページ上にスクロール |
| `Ctrl-f` | 1ページ下にスクロール |
| `g` | バッファの先頭に移動 |
| `G` | バッファの末尾に移動 |

## プラグイン関連

### tmux-resurrect / tmux-continuum

- **自動保存**: 15分ごとに自動保存
- **自動復元**: tmux起動時に自動復元
- `Prefix Ctrl-s`: セッションを手動保存
- `Prefix Ctrl-r`: セッションを手動復元

### tmux-which-key

- `Prefix Space`: キーバインドのヘルプを表示

## マウス操作

- マウスが有効化されています
- ペインのクリックで選択
- ペインの境界をドラッグしてリサイズ
- マウスホイールでスクロール
- ドラッグで選択（コピーモードに自動的に入る）

## Tips

1. **ネストされたtmuxセッション**: `Prefix Prefix <key>` でネストされたセッションにコマンドを送信
2. **コマンドラインからペイン作成**: `tmux split-window -h` (垂直分割)
3. **セッション名の変更**: `tmux rename-session -t old-name new-name`
4. **すべてのペインで同じコマンド実行**: `Prefix :setw synchronize-panes on`
