# AeroSpace チートシート

AeroSpaceはmacOS用のタイリングウィンドウマネージャーです。

## 基本設定

- **プレフィックス**: `Alt` (Option)
- **レイアウト**: tiles (デフォルト) / accordion

## レイアウト切り替え

| キーバインド | 説明 |
|------------|------|
| `Alt /` | tiles レイアウトを水平/垂直で切り替え |
| `Alt ,` | accordion レイアウトを水平/垂直で切り替え |

## ウィンドウフォーカス

| キーバインド | 説明 |
|------------|------|
| `Alt h` | 左のウィンドウにフォーカス |
| `Alt j` | 下のウィンドウにフォーカス |
| `Alt k` | 上のウィンドウにフォーカス |
| `Alt l` | 右のウィンドウにフォーカス |

## ウィンドウ移動

| `Alt Shift l` | ウィンドウを右に移動 |

## ウィンドウ操作

| キーバインド | 説明 |
|------------|------|
| `Alt Shift f` | フルスクリーン切り替え |
| `Alt Shift Tab` | ワークスペースを次のモニターに移動 |

## リサイズモード

| キーバインド | 説明 |
|------------|------|
| `Alt Shift r` | リサイズモードに入る |

### リサイズモード内のキーバインド

| キーバインド | 説明 |
|------------|------|
| `h` | 幅を -50 リサイズ |
| `j` | 高さを +50 リサイズ |
| `k` | 高さを -50 リサイズ |
| `l` | 幅を +50 リサイズ |
| `b` | すべてのウィンドウサイズをバランス調整 |
| `-` | スマートリサイズ -50 |
| `=` | スマートリサイズ +50 |
| `Enter` / `Esc` | メインモードに戻る |

## サービスモード

| キーバインド | 説明 |
|------------|------|
| `Alt Shift ;` | サービスモードに入る |

### サービスモード内のキーバインド

| キーバインド | 説明 |
|------------|------|
| `Esc` | 設定をリロードしてメインモードに戻る |
| `r` | ワークスペースツリーをフラット化（レイアウトリセット） |
| `f` | floating/tiling レイアウトを切り替え |
| `Backspace` | 現在のウィンドウ以外をすべて閉じる |
| `Alt Shift h` | 左のウィンドウと結合 |
| `Alt Shift j` | 下のウィンドウと結合 |
| `Alt Shift k` | 上のウィンドウと結合 |
| `Alt Shift l` | 右のウィンドウと結合 |

## 自動ルール

設定済みの自動ルール:
- **Arc Browser** (`company.thebrowser.Browser`): 自動的にワークスペース B に配置
- **Google Chrome** (`com.google.Chrome`): 設定継続中

## マウス動作

- フォーカスしたモニターが変更されると、マウスが自動的に中央に移動
- フォーカスしたウィンドウが変更されると、マウスが自動的にウィンドウ中央に移動

## Tips

1. **ワークスペース命名規則**: アルファベットワークスペースは用途別に命名すると管理しやすい
   - B: Browser, C: Communication, D: Development, M: Music, N: Notes, T: Terminal など
2. **モニター間移動**: `Alt Shift Tab` で現在のワークスペース全体を次のモニターに移動
3. **レイアウトリセット**: サービスモードで `r` を押すとレイアウトがリセットされる
4. **設定変更**: `~/.aerospace.toml` を編集後、サービスモード (`Alt Shift ;`) から `Esc` でリロード
